Student Marks Analyser 

##Overview 
This is a beginner-friendly data analysis project built using Python and NumPy. It analyses student marks and provides useful insights like average, mean marks and highest marks.

##Features :
- Stores student names and marks.
- Calculates :
-     1. Average Marks
-     2. Toppers Marks
-     3. Mean Marks 
-     4. After analysing the marks of the students, the code generates a graph depicting the performance of the class.
- Identifies top performing student.
- Language used is Python.
- Tech stack comprises NumPy and Matplotlib.
